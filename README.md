# Sales Dashboard (Power BI)

## Overview

This project is a sales dashboard built using Power BI to analyze sales performance.

## Features

* Total Sales, Profit, Quantity KPIs
* Sales trend analysis
* Category-wise sales comparison
* Region-wise distribution

## Tools Used

* Power BI
* Excel

## Files

* SalesDashboard.pbix
* sales_data.xlsx
* Report.pdf
